# Consumer Data Analysis

## Project Overview

This project analyzes consumer shopping behavior to understand purchasing trends, customer segments, and revenue patterns.

The analysis pipeline includes:

* Data cleaning and exploration using Python
* SQL analysis using MySQL
* Interactive dashboard creation using Power BI

The goal of this project is to demonstrate a complete **data analytics workflow from raw data to business insights**.

---

## Dataset

The dataset contains customer shopping information including:

* Customer ID
* Age
* Gender
* Item Purchased
* Category
* Purchase Amount (USD)
* Location
* Size and Color
* Season
* Review Rating
* Subscription Status
* Shipping Type
* Discount Applied
* Previous Purchases
* Payment Method
* Frequency of Purchases


---

## Tools & Technologies

* Python
* Pandas
* MySQL
* SQL
* Power BI

---

## Project Workflow

### 1. Data Cleaning (Python)

Steps performed:

* Loaded dataset using Pandas
* Checked structure using head(), info(), and describe()
* Handled missing values
* Standardized column names
* Created Age Groups using quantiles
* Converted purchase frequency into numeric days
* Removed redundant columns

---

### 2. SQL Analysis (MySQL)

The cleaned dataset was stored in MySQL and analyzed using SQL queries.

Some business questions explored:

* Total revenue generated by male vs female customers
* Customers who used discounts but spent more than average
* Spending comparison between subscribed and non-subscribed customers
* Customer segmentation based on previous purchases
* Subscription behavior of repeat buyers
* Revenue contribution by age group


---

### 3. Power BI Dashboard

An interactive dashboard was built to visualize key insights from the dataset.

Key metrics displayed:

* Total Customers
* Average Purchase Amount
* Total Purchase Revenue
* Average Review Rating

Visualizations include:

* Revenue by Category
* Sales by Category
* Revenue by Age Group
* Sales by Age Group
* Customer distribution by Season
* Percentage of customers by Subscription Status

Dashboard filters allow users to explore data by:

* Gender
* Category
* Subscription Status
* Shipping Type

---

## Some Key Insights

* Clothing generated the highest revenue among product categories.
* Young Adult customers contributed the largest share of revenue.
* Customers without subscriptions tend to spend more overall.

These insights can help businesses improve marketing strategies and product offerings.
